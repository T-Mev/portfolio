FULL STACK PROJECT

Technologies Used:
Angular 8 | TypeScript / JavaScript | HTML5 | CSS3 | Angular Material | Bootstrap 4 | Java | Spring Boot | Hibernate | MySQL

Checkpoint is a video game library manager used to track a user's collection utilizing the IGDB API to gather all the necessary game information.
Users can search from thousands of entries, view dynamic game pages with detailed information, add or delete games from their collection, view upcoming titles, and share their collections with others. Uses Angular for the front-end that communicates with a Java / Spring Boot back-end that was designed as a REST API in order to manage user and IGDB API data. Users and their collections are saved using a MySQL database.
